Welcome to XMLUnit's new Home

The development of XMLUnit 2.x - both for Java and .NET is in the process of moving to github.

XMLUnit for Java 1.x still has its home at sourceforge, if you want to report bugs, please use the issue tracker over there - feature request should go against the 2.x version here.

Currently the code lives in https://github.com/xmlunit/xmlunit - a single repository that shares the code for Java and .NET, the user's guide will be created inside a github wiki.

Right now the focus is on migrating content and creating a viable roadmap towards the first release of XMLUnit 2.0.
